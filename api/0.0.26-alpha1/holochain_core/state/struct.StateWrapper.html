pub struct StateWrapper { /* fields omitted */ }

This type wraps (decorates) InnerState with an option and re-exports and delegates all methods of InnerState. It owns the InnerState and keeps it in a Option so that it can be dropped explicitly. It also adds a function drop(&mut self) which sets the option to None which will drop the InnerState.
